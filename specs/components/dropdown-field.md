# DropdownField

## Purpose

Exists as a UI component to display the selected data, then when in edit model it will operate as a dropdown to choose new data to display. Once a selection is made and confirmed an update is made in the database.

## Layout

There will be a label above, a text field below, and a pencil icon next to the text field. When in edit mode, the textfield will become a dropdown. Next to the dropdown there will be a check button and an x button. 

## Functions

- Edit Button will change the text field to a dropdown and display the check and x button
- X button will cancel the opperation
- Check button will approve and update the database

## Notes

This will work similar in style and function to the EditableField component.
