# Admin Approve Character

## Purpose

This page exists to allow admins to see relevant information and approve characters for use in the game. 

## Layout

This layout will display a few sections that the admin will need to make approvals. 

## Sections

- Character Info - This will show the player name, the player discord, the player email, the character name, the bloodline, and the kingroup, character submission (created at) date (mm/dd/yy)
- Character Backstory - This will be a text field that displays a formated version of the player's backstory.
- Players Characters - A list of characters that already belong the the player_id for this character
- Approve - An editable XP field and an approve button.
