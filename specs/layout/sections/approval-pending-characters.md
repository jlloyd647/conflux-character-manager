# Approval Pending Character Section

## Purpose

A component that gives a list of current characters that require admin approval.

## Layout

This section will display a list of unapproved characters in a table with relevant columns. When the row is clicked, it should take the admin to the admin/characters/*/approve page. * being the character ID

## Columns

- Player Name
- Character Name
- Bloodline
- Kingroup
- Date of submission (MM/DD/YY)
- New Player Liason Contact Method
