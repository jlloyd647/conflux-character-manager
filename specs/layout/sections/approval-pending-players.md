# Approval Pending Character Section

## Purpose

A component that gives a list of current players that require admin approval.

## Layout

This section will display a list of unapproved players in a table with relevant columns. When the row is clicked, it should take the admin to the admin/players/*/approve page. * being the character ID

## Columns

- Player Name
- Player Email
- Player Discord
- Date of submission (MM/DD/YY)
- New Player Liason Contact Method

## Features

When a player is approved, it will set their status to "Active"
