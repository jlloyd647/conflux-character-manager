## About Reference Specifications

Reference specifications define facts and serve as a source of truth. Their purpose is to document structures, definitions, allowed values, relationships, and other information that should be referenced rather than debated or reinterpreted. These documents should remain descriptive rather than explanatory. When asking "What exists?" or "What is the current definition of this?", the answer should be found in a reference specification.
